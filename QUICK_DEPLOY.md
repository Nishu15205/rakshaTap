# ⚡ Quick Vercel Deploy - 2 Steps Only!

## Method 1: With Vercel Postgres (Recommended)

### Step 1: Create Database
1. Open Vercel Dashboard: https://vercel.com/dashboard
2. Go to **Storage** tab
3. Click **Create Database** → **Postgres**
4. Choose any region → **Create**
5. ✅ Done! DATABASE_URL auto-added!

### Step 2: Deploy
- Import repo: https://github.com/Nishu15205/rakshaTap
- Click **Deploy**
- ✅ Done!
